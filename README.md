# Features
 - Markdown and Iframe descriptions
 - Minimalistic website (with dark theme)
 - Bot resubmit option
 - Bot widget
 - Search option
 - Easy setup (all customisation features in one file)
 - Discord bot for bot verification
 - Server count API
